<p>GitHub, the popular developer platform owned by Microsoft, confirmed it was hacked and attackers had stolen data from around 3,800 internal code repositories.</p>
<p>The code hosting and sharing giant said <a href="https://x.com/github/status/2056884788179726685">in a series of posts on X</a> that it has “no evidence of impact to customer information stored outside of GitHub’s internal repositories,” but noted its investigation was ongoing. GitHub said it “detected and contained a compromise of an employee device involving a poisoned VS Code extension,” referring to a plugin for Visual Studio Code, a popular code editor that developers use for programming.</p>
<p>Hackers are increasingly targeting popular open-source projects, including coding extensions, with the aim of compromising developers’ computers and their projects. Targeting popular projects allows hackers to gain access to vast numbers of computers at the same time, magnifying the impact of their attacks.</p>
<p>GitHub did not name the compromised extension.</p>
<p><a href="https://therecord.media/github-confirms-teampcp-hack-customers-unaffected">The Record</a> and <a href="https://www.bleepingcomputer.com/news/security/github-confirms-breach-of-3-800-repos-via-malicious-vscode-extension/">Bleeping Computer</a> report that a hacking group called TeamPCP has taken credit for the GitHub breach, and is selling the data on a cybercrime forum.</p>
<p>GitHub did not immediately respond to a request for comment about the incident, or answer questions on whether it has received any communication from the hackers, such as a demand for ransom.</p>
<p>TeamPCP previously claimed credit for a data breach at the European Commission that resulted in the theft of more than 90 gigabytes of data from the cloud storage of the <a href="https://techcrunch.com/2026/04/03/europes-cyber-agency-blames-hacking-gangs-for-massive-data-breach-and-leak/">EU’s executive arm</a>. The hackers had stolen the European Commission’s cloud key during an earlier breach at Trivy, a vulnerability scanning tool, by pushing <a href="https://techcrunch.com/2025/04/25/techcrunch-reference-guide-to-security-terminology/#infostealers">info-stealing malware</a> to Trivy’s downstream users.</p>
<p>OpenAI was also <a href="https://techcrunch.com/2026/05/14/openai-says-hackers-stole-some-data-after-latest-code-security-issue/">targeted recently in a similar but separate attack</a> that saw hackers break into Tanstack, a platform used by web developers, to push updates containing malware that let the hackers steal passwords and tokens from users.</p>
