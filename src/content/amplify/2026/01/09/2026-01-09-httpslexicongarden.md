<p>Tracking <strong>666</strong> lexicons from <strong>80</strong> identities | <strong>6</strong> examples for <strong>4</strong> lexicons | <strong>478</strong> schema relationships</p>
<p>Lexicon Garden is a discovery platform for <a href="https://atproto.com/specs/lexicon">ATProtocol lexicon schemas</a>. Lexicons define the data structures and API endpoints used by applications in the ATProtocol ecosystem, including Bluesky. This site indexes lexicon schemas published to the network, making them easy to browse, search, and understand.</p>
