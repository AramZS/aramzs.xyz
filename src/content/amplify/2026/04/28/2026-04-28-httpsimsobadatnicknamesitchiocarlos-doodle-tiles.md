<p>Tileset for use with Hex Kit, but might be usable with other hex-based mapping software that allows for importing tiles. Mostly made for my own personal use since I haven't found a Hex Kit tileset that includes all the things I generally tend to use in my maps, but I thought other people might like it too.</p>
<p>To use it unzip the file and then go to Hex Kit, from the menu click on File &gt; Import Tiles, and then choose the folder containing the tileset.</p>
