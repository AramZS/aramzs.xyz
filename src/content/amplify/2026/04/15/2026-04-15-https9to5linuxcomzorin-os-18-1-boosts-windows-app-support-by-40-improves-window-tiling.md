<p>Zorin OS 18.1 is out today as the first minor update to the latest <strong><a href="https://9to5linux.com/zorin-os-18-officially-released-specifically-tailored-for-windows-10-expats">Zorin OS 18</a></strong> series of this Ubuntu-based distribution that aims to convert more Windows users to switch to Linux due to its friendly look and feel.</p>
<p>Zorin OS 18.1 comes six months after Zorin OS 18 with an up-to-date base from the <strong><a href="https://9to5linux.com/ubuntu-24-04-4-lts-is-now-available-for-download-powered-by-linux-kernel-6-17">Ubuntu 24.04.4 LTS (Noble Numbat)</a></strong> operating system, featuring the Linux 6.17 HWE (Hardware Enablement) kernel and Mesa 25.2 graphics stack, as well as the latest <strong><a href="https://9to5linux.com/libreoffice-26-2-open-source-office-suite-officially-released-this-is-whats-new">LibreOffice 26.2</a></strong> office suite.</p>
<p>This release bumps the suggested alternatives for Windows apps by more than 40 percent to a total of 240 applications. This is one of Zorin OS’s niche features, recommending users tailored alternatives to sideloading their Windows executables.</p>
<p>“If you’re migrating from Windows, this means you’ll get tailored guidance on how to get the most compatible versions of your favourite apps in Zorin OS. It’s one of many reasons why Zorin OS is the ideal choice for Windows users making the switch,” said the Zorin OS team.</p>
<p>Zorin OS 18.1 also brings various improvements to the desktop in the form of new options to the Advanced Window Tiling feature, including the ability to bring all tiled windows to the foreground together when switching to a tiled app from the taskbar or dash.</p>
<p>In addition, the Advanced Window Tiling feature now supports enhanced edge tiling, allowing you to snap windows to your active custom tiling layout instead of only screen half and quarter tiling, as well as to reorder the tiling layouts from the Edit Layouts dialog in the Advanced Window Tiling settings.</p>
<p>The panel has been updated as well to work seamlessly with right-to-left (RTL) scripts, and a new “Tray Icons in the Panel” toggle was added to show app icons in the system tray. There are also many under-the-hood performance improvements that should make Zorin OS 18.1 feel a bit faster and smoother.</p>
<p>Alongside the launch of Zorin OS 18.1, the team also brought back the Zorin OS Lite edition, which features the <strong><a href="https://9to5linux.com/xfce-4-20-desktop-environment-released-with-experimental-wayland-support">Xfce 4.20</a></strong> desktop environment, fingerprint reader support, seamless integration with Web Apps, updated desktop themes, and a redesigned file manager.</p>
<p>Zorin OS 18.1 is available for download right now from the <em><a href="https://zorin.com/os/download/">official website</a></em> as Core and Lite editions, as well as a commercial-only Pro edition. Being based on <strong><a href="https://9to5linux.com/ubuntu-24-04-lts-users-get-linux-6-17-and-mesa-25-2-ahead-of-ubuntu-24-04-4-lts">Ubuntu 24.04 LTS</a></strong>, the Zorin OS 18 series will continue to get support with software updates and security patches until June 2029.</p>
<p>Existing Zorin OS 18 users can upgrade to Zorin OS 18.1 by installing the latest updates from the Software Updater or by running the <code>sudo apt update &amp;&amp; sudo apt full-upgrade</code> commands in the Terminal app. Zorin OS 17 users can also update to Zorin OS 18 by following <em><a href="https://help.zorin.com/docs/getting-started/upgrade-zorin-os/">this tutorial</a></em>.</p>
