<p>Markdown powers so much of our documentation, blogs, notes, and READMEs. But let’s face it: most editors treat it like plain text in a glorified notepad with a preview pane tacked on. It’s functional… but uninspiring. Side-by-side views get cluttered, the experience feels static, and it rarely helps you <em>think</em> better.</p>
<p>R<a href="https://dev.to/vasughanta09/why-i-stopped-writing-markdown-in-vs-code-and-what-i-built-instead-2g8e"><em>ead the full artcile on dev.to</em></a></p>
<p>I got tired of that. I didn’t just want to write Markdown — I wanted a real creative space for ideas. So instead of endlessly tweaking VS Code extensions, I built something different: <strong>Markdown Weaver</strong>.</p>
<p>Today I’m excited to share that <strong>Markdown Weaver v1.3.0</strong> is live — and it takes Markdown writing way beyond basic editing into a full visual thinking + presentation studio.</p>
<p>Here’s exactly why many of us (including me) are ready to move on from classic VS Code Markdown workflows — and what makes this new tool worth trying.</p>
<h2>1. See Your Ideas as a Living Mind Map (Not Just Text)</h2>
<p>Regular editors show you words. Markdown Weaver shows you structure.</p>
<p>Hit one button and your entire document turns into an <strong>interactive mind map</strong> (built on Markmap under the hood). Headers become nodes, nested sections branch out naturally.</p>
<ul><li>Spot logical holes or weak connections at a glance during brainstorming</li><li>Click any node → jump straight to that part of your document</li><li>Export the map as clean SVG for slides, docs, or sharing</li></ul>
<p>It’s like having an instant visual outline that actually helps you reason.</p>
<h2>2. Turn Notes into Polished Presentations — Instantly</h2>
<p>Creating slides usually means copying text into PowerPoint or Keynote, fighting with layouts, and wasting hours.</p>
<p>Not anymore.</p>
<p>With <strong>Presentation Mode</strong> in v1.3.0:</p>
<ul><li>Write normally in Markdown</li><li>Use — — to mark slide breaks</li><li>Press play → beautiful, responsive slides appear instantly</li></ul>
<p>Bonus: Built-in AI can suggest smart slide divisions and generate quick summaries so you spend zero time formatting. Write once, present anywhere.</p>
<h2>3. Command Palette = Instant Productivity</h2>
<p>If you’re used to VS Code or Obsidian, you’ll love this.</p>
<p>Hit <strong>Ctrl+K</strong> (or <strong>Cmd+K</strong>) and the command palette pops up — your personal command center.</p>
<ul><li>Fuzzy-search any note by title or snippet</li><li>Toggle themes, export PDF/HTML, switch modes, etc. — all keyboard-driven</li><li>Feels like a “super spot” for your whole Markdown workspace</li></ul>
<p>No more menu diving.</p>
<h2>4. Context-Aware AI That Helps Without Getting in the Way</h2>
<p>We skipped the usual sidebar chatbot gimmick.</p>
<p>Instead, highlight any text → a subtle <strong>Magic Menu</strong> appears with powerful one-click actions:</p>
<ul><li>Fix grammar and typos</li><li>Make it clearer / more concise</li><li>Shift tone (professional → casual → persuasive, etc.)</li><li>Continue writing in your voice</li></ul>
<p>It feels like a thoughtful co-writer, not a distracting bot.</p>
<h2>5. A Real Workspace, Not Just a File Opener</h2>
<p>Managing lots of Markdown files used to be chaos. Now there’s a proper <strong>Project Dashboard</strong>:</p>
<ul><li>Trash with easy restore (no more accidental permanent deletes)</li><li>Writing stats: daily streaks, total words, progress tracking</li><li>Ready-to-use templates: README.md, blog post, meeting notes, API docs, etc.</li></ul>
<p>Plus a clean onboarding flow so new projects feel welcoming.</p>
<h2>Under the Hood (v1.3.0 Technical Wins)</h2>
<ul><li>Upgraded to <strong>TypeScript 5.8</strong> for better reliability</li><li>True project-based local storage (no more single-file chaos)</li><li>Real-time sync across browser tabs</li><li>Drag-and-drop any .md file to import instantly</li><li>100% offline-first — your content stays private and loads lightning-fast</li></ul>
<h2>Ready to Make Markdown Writing Enjoyable Again?</h2>
<p>Markdown Weaver is completely <strong>free</strong>, <strong>open-source</strong>, and built for writers who love Markdown’s elegance but want modern superpowers.</p>
<p>Give it a spin — I think you’ll find it hard to go back to plain old editors.</p>
