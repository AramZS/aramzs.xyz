<p>Google Ads is letting some advertisers add negative keyword lists to their Performance Max campaigns. This is despite Google's very own <a href="https://support.google.com/google-ads/answer/15726455?hl=en">help documentation</a> specifically saying, "Negative keyword lists aren’t available to use in your Performance Max campaigns."</p>
<p>I spotted this via Adriaan Dekker who posted on <a href="https://www.linkedin.com/posts/adriaan-dekker-google-ads-freelancer-rotterdam_jeremy-young-alyssa-knab-saw-that-you-can-activity-7351611713605042176-00PE/">LinkedIn</a> and credited <a href="https://www.linkedin.com/posts/jeremy-young-google-ads_googleads-ppc-activity-7351543535214469122-4UK6/">Jeremy Young</a> and <a href="https://www.linkedin.com/in/ACoAABQSc24B7-0CuO7OXLjhOPfi4qlPQbgP7nI">Alyssa Knab</a> for spotting this.</p>
<p>He shared this screenshot showing these negative keyword lists in the PMax campaigns:</p>
<figure><a href="https://images.seroundtable.com/google-ads-pmax-negative-keywords-1752846293.jpg"><img alt="Google Ads Pmax Negative Keywords" src="https://images.seroundtable.com/google-ads-pmax-negative-keywords-1752846293.jpg"/></a><figcaption><a href="https://images.seroundtable.com/google-ads-pmax-negative-keywords-1752846293.jpg">Google Ads Pmax Negative Keywords</a></figcaption></figure>
<p>As Adrian pointed out, "I think they should not work, because the documentation says the are not available, but in this example it looks like they are..."</p>
