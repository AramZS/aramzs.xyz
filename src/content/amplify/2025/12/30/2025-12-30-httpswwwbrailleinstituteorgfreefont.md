<h2>Legible. Life Changing.</h2>
<p>Named after Braille Institute founder, J. Robert Atkinson, the Atkinson Hyperlegible<sup>®</sup> font focuses on letterform distinction.</p>
<p>When we designed it in 2019, the Atkinson Hyperlegible font won <em>Fast Company</em>’s Innovation by Design Award. In 2024, it was officially added to the Cooper Hewitt, Smithsonian Design Museum’s permanent collection.</p>
<p>With more and more people facing vision impairment every year, the opportunity to make a difference in the world is profound.</p>
<p>“I’ve been using this font as my browser default since it was released and I can confidently say it’s improved my life…</p>
<p>Finally, I feel I can actually browse the web without getting a migraine every thirty minutes, so thank you so much!”</p>
<h2>Unique Design Features</h2>
<p>For low-vision readers, certain letters and numbers can be hard to distinguish from one another. The Atkinson Hyperlegible<sup>®</sup> font uses special design principles to differentiate characters and make each one unique.</p>
<p><strong>Unambiguous Letterforms</strong> Letters have received special care to ensure they’re legible and clear.</p>
<p><strong>Clear Uprights</strong> Similarly-shaped upright letters are distinctly different.</p>
<h2>One Family. Three Versions.</h2>
<p>The hyperlegible<sup>®</sup> family of fonts is now available in three versions: Atkinson Hyperlegible<sup>®</sup>, Atkinson Hyperlegible Next<sup>®</sup>, and Atkinson Hyperlegible Mono<sup>®</sup>. Not sure what you’re looking for? Compare the versions below.</p>
<p>Atkinson Hyperlegible Mono is an entirely new typeface inspired by the Atkinson Hyperlegible font. The monospaced font features characters that each occupy the same amount of horizontal space, allowing for them to be scanned quickly in table-based and coding environments. A variable version is also available.</p>
<p>Variable weight—Extrabold through Light, upright and italic.</p>
<h2>Installation Instructions</h2>
<p><b>One Typeface—Three Styles</b><br/>
The Atkinson Hyperlegible font comes in three choices: Atkinson Hyperlegible Next, Atkinson Hyperlegible Mono, and the original Atkinson Hyperlegible. We recommend installing Atkinson Hyperlegible Next to start—it’s our latest and greatest, and suitable for everyday readers.</p>
<p><strong>Seven Weights, and Italics</strong><br/>
The Atkinson Hyperlegible Next font comes in seven weights: ExtraLight, Light, Regular, Medium, SemiBold, Bold, and ExtraBold, each with an upright and Italic version. You can install all, or select which weights you want to use.</p>
<p><strong>Step 1: Download the Font File</strong><br/>
Download the ZIP file and save it to your computer. After it is downloaded, open the ZIP folder to begin the installation process.</p>
<p><strong>Step 2: Finding the Right Fonts</strong><br/>
All of the files you will need to begin to use the font are inside the main area of the unzipped folder. Make sure you’re only installing the OpenType files (.otf). The folder “For Professional Use Only” is where you will find the variable font, web fonts, and source files.</p>
<p><strong>Step 3A: Installing on a Windows PC</strong><br/>
Double-click on the font file (or files) you want to install, and then click “Install” in the font preview window. The Atkinson Hyperlegible font will then be installed.</p>
<p><strong>Step 3B: Installing on a Mac</strong><br/>
Double-click on the file (or files) you want to install in the Finder. The Font Book application will open. Click “Install” to install the font.</p>
