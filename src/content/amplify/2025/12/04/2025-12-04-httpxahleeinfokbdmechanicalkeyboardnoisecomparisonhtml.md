<h2>Ranking of Key Switch Noise Levels</h2>
<p>Here's rank of noises of <a href="http://xahlee.info/kbd/keyboard_switch_mechanisms.html">keyboard key switches</a>. The ranking is based on practical use, not from pressing a single key. The ranking is from my own experience of using of keyboards since 1991 plus watching many internet videos.</p>
<h3>Crazy Click-Clack Loud</h3>
<ul> <li><a href="http://xahlee.info/kbd/buckling_spring_key_switch.html">Buckling Spring Key Switch</a>. Loudest.</li> <li>Cherry MX Blue switches.</li> <li>Cherry MX Red switches. MX Red is silent if you press it slowly. But the noise came from key hitting bottom.</li> </ul>
<h3>Loud</h3>
<ul> <li>Cherry MX Brown switches.</li> <li>Cherry MX Black switches.</li> <li>Gateron Red switches.</li> <li>Topre Switch 45g. e.g. <a href="http://xahlee.info/kbd/idiocy_of_happy_hacking_keyboard.html">HHKB</a></li> <li>Logitech Romer-G Tactile Switch. 〔see <a href="http://xahlee.info/kbd/romer-g_switch.html">Romer-G Switch</a>〕.</li> </ul>
<h3>Some Noise</h3>
<ul> <li><a href="http://xahlee.info/kbd/keyboards_with_topre_switch.html">Topre Switch</a> silent. e.g. <a href="http://xahlee.info/kbd/hhkb_pro_hybrid_type_s.html">HHKB Pro HYBRID Type-S, 2019</a></li> <li>Cherry MX Silent Black switches.</li> <li>Cherry MX Silent Red switches.</li> <li><a href="http://xahlee.info/kbd/niz_switch.html">Niz Switch</a> topre clone.</li> </ul>
<h3>Your Wife is Sleeping Quiet</h3>
<ul> <li>Kailh BOX Silent Pink switch. (available on <a href="http://xahlee.info/kbd/uhk_keyboard_v2.html">Ultimate Hacking Keyboard (UHK 60 V2)</a>)</li> <li>Chicklet key rubber dome scissor switches. e.g. <a href="http://xahlee.info/kbd/Microsoft_sculpt_ergonomic_keyboard.html">Microsoft Sculpt Ergonomic Keyboard</a></li> <li>linear <a href="http://xahlee.info/kbd/choc_switch.html">Kailh Choc Switch</a></li> <li>Laptop keyboards. 〔see <a href="http://xahlee.info/kbd/laptop_keyboards.html">Evolution of Laptop Keyboard</a>〕</li> </ul>
<h2>Where Does the Key Noise Came From</h2>
<p>Some switches intentionally make a noise, such as Cherry MX Blue Clicky and <a href="http://xahlee.info/kbd/buckling_spring_key_switch.html">Buckling Spring Key Switch</a></p>
<p>Most switches do not make much noise if you press them slowly.</p>
<p>The noise came from:</p>
<ul> <li>The key hitting bottom. (try hit a key hard and fast, and hear the sound hitting bottom. This sound is similar if you hit desk with your finger fast. Try it. You get knocking sound.)</li> <li>Key bouncing back. (try this: press down the key slowly all the way, then move finger side way off suddenly. The key pops back by spring, “hits top”, makes noise.)</li> <li>Key housing. For example, the spacebar. The spacebar make the loudest noise, especially those from cheap rubber dome computer keyboards. e.g. <a href="http://xahlee.info/kbd/ms_keyboard/ms_natural_keyboard_4000.html">Microsoft Natural Ergonomic Keyboard 4000</a></li> <li>Friction noise. The spring, stem, keycap, together makes noises. Especially on cheap rubber dome computer keyboard.</li> </ul>
<h2>Rubber O-Ring Noise Dampener (No Good)</h2>
<p>One way to reduce noise is to buy those rubber ring switch dampeners. You put them in each key yourself.</p>
<p>I do not recommend it. Too much trouble, also it doesn't help much.</p>
