<p><code>ratatui</code> is a <a href="https://www.rust-lang.org">Rust</a> library that's all about cooking up terminal user interfaces (TUIs). It is heavily inspired by the <code>Javascript</code> library <a href="https://github.com/yaronn/blessed-contrib">blessed-contrib</a> and the <code>Go</code> library <a href="https://github.com/gizak/termui">termui</a>.</p>
<p>See the <a href="https://github.com/ratatui/ratatui">repository</a> for more information and usage examples!</p>
