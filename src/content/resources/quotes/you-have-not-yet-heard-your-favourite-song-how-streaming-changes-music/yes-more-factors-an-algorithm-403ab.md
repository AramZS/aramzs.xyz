> yes, the more factors an algorithm incorporates, the more complex the consent, transparency and control problems become, and I think that is absolutely fine. The more control we want to impose over people’s experiences, the more work we should have to do. If we aren’t willing to do it, we should stick to less complicated methods with fewer inputs. Maybe we should do that anyway.
